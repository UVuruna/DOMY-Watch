# Egyptian Gods — the Theban Weekday Theme (Planetary + Pantheon, bronze + colored)

The Egyptian set now carries BOTH rosters side by side, per the
doctrine in `research/pantheon_catalog.md` and the seat table in
`ROSTER.md`: **Planetary** (Ra, Khonsu, Montu, Thoth, Amun, Hathor,
Osiris, dual Afu-Ra — the shipped day-ruler set) and **Pantheon** (Ra,
Isis, Horus, Thoth, Anubis, Bastet, Osiris, dual Afu-Ra shared, ninth
the Pharaoh — the max-coverage roster, seated by each figure's real
mythological rank). This sheet is the ONE complete generation
reference for the theme: the Planetary bronze + colored plates
(unchanged, regenerated verbatim from `Database/symbolism.json`) plus
the new Pantheon bronze + colored plates, and the superseded Ninth.

**Border identity:** a bronze ring carved as a continuous
**hieroglyph cartouche band**, running clockwise, broken by four
roundels alternating an **ankh** and a **scarab** in relief, each
cradling the day's planetary glyph — distinct from every other bronze
theme's border (no meander, no knotwork, no paw-prints). Both rosters
share this identity; only the center figure changes between a
Planetary and a Pantheon plate.

**Style note:** aged bronze relief is the base register because this
is the metal-swap family — gold and silver variants hue-swap the same
warm bronze pixels programmatically, so no separate gold/silver
prompts are needed here. The colored register (Theban tomb-fresco,
lapis and gold) is a fresh full-color painting of the same
composition, not a recolor of the bronze plate — true for both
rosters.

## The Ninth — The Pharaoh (supersedes Set)

Owner's proposal ACCEPTED and seconded (`research/pantheon_catalog.md`,
2026-07-15): the Pharaoh — Tutankhamun's likeness, named simply
"Pharaoh" — is the Ninth for BOTH modes at once (Planetary and
Pantheon share one ninth per theme; two mode-dependent ninths would
break the Encyclopedia's honesty, per the doctrine). This
**SUPERSEDES the former Set ninth** recorded below for history — do
**NOT** generate `set.png`, in `pantheon/` or anywhere else. Set was
never actually rendered (confirmed absent from
`assets/weekday/gemini/egypt/primary/`; it was carried only as an
open line in `ROSTER.md`'s shortage list) — nothing to delete on
disk, only the spec is retired. Set's murder-of-Osiris story survives
regardless, inside the Osiris and Horus articles; Set himself retires
to the Encyclopedia's Wider-Pantheon lane (non-seated, no dial role).

Why the Pharaoh clears the never-outranked bar the Ninth doctrine
requires: the living king was Horus in life and Osiris in death — the
throne above and the throne below united in ONE living body, which is
the Union form word for word — and the golden death mask is arguably
the single most recognizable Egyptian image on Earth, so no seated
god's plate can outrank it. Read this against Set's old framing
below: where Set stood for the excluded murderer-brother (a real and
still-valid Egyptian archetype, just not this theme's ninth anymore),
the Pharaoh stands for the opposite pole the doctrine now prefers —
not exclusion but union, not a lesser exile but the culture's own
center.

The bronze prompt is listed under "Pantheon — bronze plates" above
(`assets/weekday/egypt/pantheon/pharaoh.png`); the colored prompt under "Pantheon —
colored plates" above (`assets/weekday/egypt/pantheon/colored/pharaoh.png`). Both are
listed exactly once each in this sheet — not reproduced here to keep
one prompt block per plate.

read Set as the excluded brother-murderer against Osiris's passive
